create a new repository on the command line...

echo "# git-github-estudo-2025-01" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/marcusvbrangel/git-github-estudo-2025-01.git
git push -u origin main


or push an existing repository from the command line...

git remote add origin https://github.com/marcusvbrangel/git-github-estudo-2025-01.git
git branch -M main
git push -u origin main
